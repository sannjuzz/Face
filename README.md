Description:
AI Face Emotion Persona Overlay is a real-time computer vision project that detects faces, recognizes their emotions, and overlays a personalized visual persona or avatar. This project uses AI and Machine Learning to create interactive, dynamic visual feedback based on the user's emotions.

Features:

Real-time face detection using OpenCV

Emotion recognition (happy, sad, angry, surprised, neutral, etc.)

Persona overlay for each detected emotion

Live camera feed visualization

Lightweight and fast

Tech Stack:

Python

OpenCV

MediaPipe

TensorFlow / Keras (for emotion recognition model)

NumPy / Pandas


Usage:
Run the main Python file:
python main.py
